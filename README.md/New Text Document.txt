# 📦 Supply Chain Logistics EDA

## 🚀 Project Overview

Performed exploratory data analysis on a supply chain dataset to identify key drivers of freight cost and improve logistics efficiency.

## 📊 Dataset

* 10,000+ shipment records
* Includes vendor, shipment mode, freight cost, and product details

## 🧹 Data Cleaning

* Converted data types
* Handled missing values
* Removed duplicates

## 📈 Key Insights

* Freight cost is highly skewed
* Shipment mode significantly impacts cost
* Some vendors are consistently expensive

## 💡 Business Recommendations

* Optimize shipment mode usage
* Reduce dependency on costly transport methods
* Improve vendor selection strategy

## 🛠️ Tools Used

Python, Pandas, NumPy, Seaborn, Matplotlib
