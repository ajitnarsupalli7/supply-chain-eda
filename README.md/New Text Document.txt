# 📦 Supply Chain Logistics EDA

## 🚀 Project Overview

This project analyzes a supply chain dataset to identify key drivers of freight cost and improve logistics decision-making.

---

## 📊 Dataset

* 10,000+ shipment records
* Includes vendor, shipment mode, freight cost, and product details

---

## 🧹 Data Cleaning

* Converted data types (Freight Cost, Weight)
* Handled missing values
* Removed duplicate records

---

## 📈 Key Insights

* Freight cost is highly skewed with outliers
* Shipment mode is the biggest cost driver
* Certain vendors have consistently higher costs

---

## 💡 Business Recommendations

* Optimize shipment mode selection
* Reduce dependency on high-cost transport
* Improve vendor selection

---

## 🛠️ Tools Used

Python, Pandas, NumPy, Seaborn, Matplotlib
